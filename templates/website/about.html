About Notes App

A beginner-friendly Django project built to understand backend development through real-world structure and clean design.

📌 Project Overview

Notes App is a beginner-friendly Django project designed to help understand backend development. It allows users to create, view, and manage notes in a simple and structured way.

🎯 Purpose

The main purpose of this project is to practice Django concepts like URLs, Views, Templates, and Database integration in a real-world style workflow.

🚀 What You Will Learn

Django project structure
Request and response cycle
Templates system
Static files usage
CRUD operations (coming soon)
